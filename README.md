# MyCourses Dark Theme
A dark theme for MyCourses and some scripts to reorganize/delete parts.

## MyCourses.css Features
- Makes MyCourses backgrounds dark
- Deletes annoying 'Find My Courses' text box and the user links box on the right 

## MyCourses.js Features
- Deletes annoying 'Find My Courses' text box
- *Work In Progress* allow spell-check for quizzes
